# Sales-Performance-Dashboard-Powerbi
![Sales Dashboard](Sales_Dashboard.png)
Power BI dashboard for analyzing sales trends, profit, orders, and customer behavior in an e-commerce dataset.
Sales Performance Dashboard

Tools Used
Power BI
Power Query
DAX

Project Overview
This project presents an interactive Power BI dashboard designed to analyze e-commerce sales performance across different regions, product categories, and customer segments.

Key Metrics
• Total Sales: 5.87M
• Total Profit: 970K
• Total Orders: 35K
• Quantity Sold: 51K
• Average Order Value: 170

Dashboard Insights
• Monthly sales trend analysis
• Regional sales comparison
• Category-wise sales distribution
• Payment method usage analysis
• Customer gender distribution

Features
• Interactive region filter
• KPI summary cards
• Dynamic visualizations for trend analysis
